Overview

In this assignment, you will create a simple Unity scene that includes basic objects with material, lighting, skybox and camera animation.

What you need to do

This assignment is divided into five major tasks and a extra one, which are described below according to the order we cover it in the class.

Task 1: Creating basic 3D objects
Your warm up task is to create basic 3D objects which we showed in our class.
Use at least a cube, a sphere and a plane to build up your scene.

We encourage you to use more objects to build a scene and change their transform arguments in the inspector in order to get familiar with future tasks.

Extra Credit:
Use Terrain editor to create your own world.

Task 2: Adding light for the scene
The default Unity scene is with a directional light. Change the argument of it to give your scene a nice overall lighting.
Adding at least TWO point lights with different colors.
Change the skybox. We provide you with a unity package. You can also find free ones in Asset Store.
Extra Credit:
Using a spot light to enhance the atmosphere.

Task 3: Create materials for your objects
Create materials for your object.
Include at least one diffuse one and a reflective one.
Add Texture to at least one of them.

Extra Credit:
Create material Glass
Try normal maps and emission maps.

Task 4: Create an animation for camera
Create an animator controller to your main camera.
Use unity animating tool to make a easy animation for it.

Extra Credit:
(Note: if you are doing this one, make any animation to an object instead of camera.)
Attached are references for useful built-in functions for this task. Read them and try to write a controller that allows your camera to move around according to inputs.
http://docs.unity3d.com/ScriptReference/Transform-position.html
http://docs.unity3d.com/ScriptReference/Transform.Translate.html
http://docs.unity3d.com/ScriptReference/Transform.LookAt.html
http://docs.unity3d.com/ScriptReference/Input.GetKey.html
http://docs.unity3d.com/ScriptReference/KeyCode.html

Task 5: Build your scene
Following the steps we showed on class, you can build your scene now.
You can leave the settings default ones.

NOTE: You can always go to http://docs.unity3d.com/Manual/index.html and search for help.
